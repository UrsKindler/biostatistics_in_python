# 1. Data Quality Assessment

  ### Overview
- **Purpose**: Evaluate data completeness, identify outliers, and assess data integrity before statistical analysis
- **Use Case**: Initial data exploration, missing data pattern identification, outlier detection
- **Prerequisites**: Raw dataset with mixed data types
- **Data Types**: Any data type (numeric, categorical, text, dates)

### When to Use
#### Decision Criteria
- **Use always**: As first step in any data analysis workflow
- **Essential when**: High-dimensional data, clinical/biological datasets with expected missing patterns
- **Critical for**: Datasets from multiple sources or collection timepoints
- **Don't skip when**: Working with real-world messy data

### R vs Python 

| Aspect | R | Python |
|--------|---|--------|
| Package | `VIM` | `missingno`, `pandas` |
| Missing Pattern | `VIM::aggr(data)` | `missingno.matrix(df)` |
| Data Summary | `summary(data)` | `df.describe()` |
| Outlier Detection | `boxplot.stats(data)$out` | `df.quantile()` method |
| Data Structure | `str(data)` | `df.info()` |

### Quick Start Code

**R Example:**
```r
# Load required libraries
library(VIM)

# Basic data inspection (base R)
summary(data)
str(data)

# Missing data visualization  
aggr(data, col=c('navyblue','red'), numbers=TRUE, sortVars=TRUE)

# Outlier detection (base R)
outliers <- boxplot.stats(data$variable)$out

print(outliers)
```

**Python Example:**
```python
import pandas as pd
import missingno as msno

# Load and inspect
df = pd.read_csv('data.csv', index_col=0)
print(df.info())
print(df.describe())

# Missing data visualization
msno.matrix(df)

# Simple outlier detection using pandas only
Q1 = df.quantile(0.25)
Q3 = df.quantile(0.75)
IQR = Q3 - Q1
outliers = ((df < (Q1 - 1.5 * IQR)) | (df > (Q3 + 1.5 * IQR))).sum()
print(f"Outliers per column:\n{outliers}")
```

**Visualization**

```python
import pandas as pd
import numpy as np
import missingno as msno
import matplotlib.pyplot as plt
import seaborn as sns

# Simuliere biologische Daten 
np.random.seed(42)
n_samples = 50
n_features = 20

# Create data with realistic missingness patterns
data = np.random.randn(n_samples, n_features) * 10 + 100
# Introduce MNAR pattern (low values â†’ missing)
data[data < 95] = np.nan if np.random.random() < 0.3 else data[data < 95]

df = pd.DataFrame(data, 
                 columns=[f'Protein_{i+1}' for i in range(n_features)],
                 index=[f'Sample_{i+1}' for i in range(n_samples)])

# Create figure with 2 subplots
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 1. Missing data pattern
msno.matrix(df, ax=axes[0], sparkline=False, fontsize=10)
axes[0].set_title('Missing Data Pattern (MNAR)', fontsize=12, fontweight='bold')

# 2. Outlier detection boxplot
axes[1].boxplot([df[col].dropna() for col in df.columns[:10]], 
               labels=[f'P{i+1}' for i in range(10)])
axes[1].set_title('Outlier Detection (First 10 Proteins)', fontsize=12, fontweight='bold')
axes[1].set_ylabel('Abundance')
axes[1].set_xlabel('Protein ID')
axes[1].grid(axis='y', alpha=0.3)

plt.tight_layout()
plt.savefig('01_data_quality_visualization.png', dpi=300, bbox_inches='tight')
plt.show()
```
